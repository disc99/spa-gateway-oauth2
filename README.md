Minimal SPA with login feature.

```bash
yarn start
```

## uaa
User Account and Authentication (UAA) Server.

```bash
./gradlew bootRun
```

## gateway
Stateless gateway not dependent on Spring Security.

```bash
./gradlew bootRun
```
